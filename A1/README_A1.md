This script counts word from an input text file and returns the number of unique words and total words (format: `unique / total`)

Usage:
```
python julia-gro-WordCount.py <input_file> 
```

Parameters:
* `-l`: prints out a list, that is sorted by wordcount and word respectively (does not output unique and total word counts)
* `-I`: ignores case

Output:
python julia-gro-WordCount.py WordCount-test1.in 
    78 / 89
