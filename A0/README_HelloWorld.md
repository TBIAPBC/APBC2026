# A0 - Hello World Assignment 

Last update: 18.03.26
Responsible Author: Anna Gsteu


##  Description 

This Python script reads the content of an input file and prints a "Hello World!" message followed by the file content.


### Requirements:
	- Python 3
	
### How to run:
Run the script from the command line and provide an input file as an argument:
	python anna-gst-HelloWorld.py inputfile

### Output files 
You can save the output into a file using redirection:
	python anna-gst-HelloWorld.py inputfile > outputfile
